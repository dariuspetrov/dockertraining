Noi folosim CosmosDB Java SDK si avem mai multe api-uri, printre care si count, care este implementat folosind 
metoda „queryAggregateValues”, care intoarce un List<Object>, asa:





List<Object> queryAggrResults = getDocumentClient().queryAggregateValues(collectionLink, query, feedOptions);

Long count = 0L;

for (Object obj : queryAggrResults) {

    if (obj instanceof Long) {

        count += (Long) obj;

    } else if (obj instanceof Integer) {

        count += ((Integer) obj).longValue();

    }

}

Pentru celelalte call-uri, am acces la un obiect de tipul Response, de la care pot folosi metoda „getRequestCharge()” 
pentru a obtine exact numarul de request units consumat de acel call. Problema mea este ca nu stiu cum sa accesez
aceasta informatie folosind metoda de mai sus.
 
Scopul meu este sa loghez cate RU consuma fiecare call.
